Kavo UI Optimized

The ultimate, ultra-lightweight UI library for Roblox exploit scripts. Rewritten from the ground up for maximum performance, zero memory leaks, modern aesthetics, and powerful new features.

Getting the Loadstring

To start using the library, execute the following loadstring at the very top of your script:

Lua 
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/sdwird/KAVO/refs/heads/main/source.lua"))()

Window & Themes

Creating the Window

Initializes the main UI. You can pass a string for a default theme, or a table for a custom theme.

Lua
local Window = Library.CreateLib("Hub Title", "DarkTheme", Enum.KeyCode.RightShift)
  • Title (string): The text displayed at the top of your UI.
  • Theme (string/table): Name of a built-in theme, or a custom color table.
  • ToggleKey (Enum.KeyCode): Optional keybind to hide/show the UI. Defaults to RightShift.

Dynamic Theme Switching

Change the entire UI theme instantly without requiring a script restart.

Lua
Window:ChangeTheme("Ocean")

Safe Unload

Completely destroys the UI, disconnects all event listeners (keybinds, dragging, tracking), and clears memory to prevent lag over multiple injections.

Lua
Window:Unload()

Watermarks & Tracking New Features

Notification System

Triggers a beautiful sliding notification in the bottom right corner of the screen.

Lua
Window:Notify("Inject Success", "Successfully loaded all scripts.", 5, "Success")
  • Title (string): Header of the notification.
  • Text (string): Body content.
  • Duration (number): Time in seconds before it auto-closes.
  • Type (string): Accent color. Options: "Info", "Success", "Warning", "Error".

Watermark

Creates a draggable, floating watermark in the corner of the screen. Returns an updater function for live FPS/Ping trackers.

Lua
local WM = Window:SetWatermark(true, "My Hub | FPS: 60")

-- Update it later inside a loop:
WM.Update("My Hub | FPS: 55")

Active Modules List

Enables a draggable panel that automatically tracks and displays any Toggles the user currently has turned on.

Lua
Window:SetActiveModules(true)

Config Save & Load

The optimized library natively supports saving and loading UI states to JSON files in your workspace. To use this effectively, you must assign a Flag (a unique string identifier) when creating elements like Toggles, Sliders, Dropdowns, ColorPickers, and Keybinds.

Lua
-- Example element with a flag
Section:NewToggle("Aimbot", "Locks onto players", function(state) end, "AimbotToggleFlag")

-- Save Config (Folder Name, File Name)
Window:SaveConfig("MyHubFolder", "LegitConfig")

-- Load Config
Window:LoadConfig("MyHubFolder", "LegitConfig")

-- Reading a flag directly from your code:
local isAimbotOn = Library.Flags["AimbotToggleFlag"]

Tabs & Sections

Creating a Tab (With Icons)

Tabs appear on the left sidebar. You can optionally pass a Roblox Asset ID to display an icon next to the tab name.

Lua
local MainTab = Window:NewTab("Main Settings", "rbxassetid://3926305904")

Creating Collapsible Sections

Sections logically group your elements. Sections are now fully collapsible by clicking their header chevron.

Lua
local PlayerSection = MainTab:NewSection("Player Modifications", false)
  • Title (string): Header name of the section.
  • Hidden (boolean): If true, the colored header and collapse arrow are completely invisible.

Buttons, Toggles & Sliders

Most elements now support optional Flag (for config saving) and IconId (for custom images) parameters at the end of their arguments. If you don't need a flag but want an icon, pass nil for the flag.

Button & Toggle

Lua
Section:NewButton("Kill All", "Instantly kills all enemies", function()
    print("Clicked!")
end, "rbxassetid://123456") -- Icon is optional

Section:NewToggle("Aimbot", "Enable auto-aim", function(state)
    print("Aimbot is now:", state)
end, "AimbotTogFlag", "rbxassetid://123456")
  • Name: Display text.
  • Tooltip: Info text shown on hover.
  • Callback: Function executed when clicked/toggled.
  • Flag (Toggle Only): Unique string for config saving.
  • IconId: Optional custom Roblox image asset ID.

Precision Slider

Sliders now support precision steps, suffixes (like "%"), and manual textbox input by clicking the number.

Lua
Section:NewSlider("Field of View", "Adjust camera FOV", 120, 70, 0.5, function(value)
    workspace.CurrentCamera.FieldOfView = value
end, "FOVSliderFlag", "°")
  • Max / Min: Highest and lowest possible values.
  • Step: The increment amount (e.g., 1 for integers, 0.1 for decimals).
  • Suffix: String appended to the display value (e.g., "%" or "x").

Dropdowns & TextInputs

Searchable & Multi-Select Dropdowns

Dropdowns contain a built-in search bar to filter large lists, and can optionally allow multiple selections. We also provide a specialized NewPlayerDropdown.

Lua
-- Standard Dropdown (Single Select)
Section:NewDropdown("Teleports", "Select location", {"Spawn", "Shop"}, function(sel)
    print("Teleporting to:", sel)
end, false, "TpDropFlag")

-- Multi-Select Dropdown
Section:NewDropdown("Target Body Parts", "Where to shoot", {"Head", "Torso"}, function(selList)
    print("Targeting:", table.concat(selList, ", "))
end, true, "TargetPartsFlag")

-- Auto-Updating Player Dropdown
Section:NewPlayerDropdown("Select Player", "Targets a player", function(playerName)
    print("Selected:", playerName)
end, false, "TargetPlrFlag")
  • List (table): Array of strings to display.
  • IsMultiSelect (boolean): If true, allows clicking multiple items. The callback returns a table of selected strings instead of a single string.

Textboxes (Copyable) & Labels

Textboxes feature an integrated clipboard copy button. Paragraphs auto-size based on content length (Supports RichText markup!).

Lua
Section:NewTextBox("Command", "Run code", function(txt) 
    print("Executed:", txt)
end, "CmdBoxFlag")

Section:NewLabel("This is a simple single-line label.")

Section:NewParagraph("Welcome to the Ultimate Script. This paragraph expands automatically to fit very long texts.")

Colors & Keybinds

Color Picker

Lua
Section:NewColorPicker("ESP Color", "Change visual colors", Color3.fromRGB(255, 0, 0), function(color)
    print("Color selected:", color)
end, "EspColorFlag")
  • DefaultColor (Color3): The initial color shown before interaction.

Keybind

Lua
Section:NewKeybind("Fly Key", "Toggle flight", Enum.KeyCode.F, function()
    print("Fly Key Pressed")
end, "FlyKeyFlag")

Dynamic Element API

Every element creation function returns a table containing specific updater functions, as well as a base API to hide or destroy the element dynamically during runtime.

Lua
local myToggle = Section:NewToggle("Godmode", "Invincibility", function() end)

-- [ Element Specific Updaters ]
myToggle:UpdateToggle("New Title", true) -- (NewTitle, ForceState)
-- Note: Buttons use UpdateButton(), Sliders use UpdateSlider(), Dropdowns use Refresh() or SetOption().

-- [ Base API (Available on ALL elements) ]
myToggle:SetVisible(false) -- Temporarily hides the element from the section
myToggle:Destroy()         -- Permanently deletes the element and clears memory

The "Mega-Hub" Example

This complete, copy-pasteable script demonstrates how to combine Tabs, Sections, Watermarks, Dynamic Tracking, and the Config Save/Load system into a massive, production-ready GUI.

MegaHub.lua 
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/sdwird/KAVO/refs/heads/main/source.lua"))()
local Window = Library.CreateLib("Mega Hub Ultimate", "Midnight", Enum.KeyCode.RightShift)

-- [ 1. Core Systems & Trackers ] --
local WM = Window:SetWatermark(true, "Mega Hub | Calculating FPS...")
Window:SetActiveModules(true) -- Enables the floating "Active Toggles" list

-- Simple FPS/Ping background loop
task.spawn(function()
    local RS = game:GetService("RunService")
    local frames = 0
    RS.RenderStepped:Connect(function() frames = frames + 1 end)
    while task.wait(1) do
        WM.Update("Mega Hub V2 | FPS: " .. frames)
        frames = 0
    end
end)

-- [ 2. Combat Tab ] --
local CombatTab = Window:NewTab("Combat", "rbxassetid://3926305904")
local AimSec = CombatTab:NewSection("Aimbot Systems")

AimSec:NewToggle("Enable Aimbot", "Locks onto players automatically", function(state)
    print("Aimbot enabled:", state)
end, "AimbotToggle")

AimSec:NewPlayerDropdown("Target Player", "Select a specific player to target", function(player)
    print("Targeting:", player)
end, false, "TargetPlayer")

AimSec:NewSlider("Smoothing", "Adjust aim smoothing", 10, 1, 0.5, function(val)
    print("Smoothing set to:", val)
end, "AimSmooth", "x")

-- [ 3. Visuals Tab ] --
local VisTab = Window:NewTab("Visuals", "rbxassetid://3926307971")
local EspSec = VisTab:NewSection("ESP Settings")

EspSec:NewColorPicker("Box Color", "Change the color of ESP boxes", Color3.fromRGB(255, 0, 0), function(color)
    print("ESP Color changed to", color)
end, "EspColor")

EspSec:NewDropdown("Render Body Parts", "Choose which parts to highlight", {"Head", "Torso", "Arms", "Legs"}, function(selected)
    print("Rendering:", table.concat(selected, ", "))
end, true, "EspParts") -- 'true' enables Multi-Select!

-- [ 4. Settings & Config Tab ] --
local SettingsTab = Window:NewTab("Settings")

-- Config System
local ConfigSec = SettingsTab:NewSection("Configuration System")
ConfigSec:NewTextBox("Config Name", "Enter name to save/load", function(txt)
    print("Config name targeted:", txt)
end, "ConfigNameBox")

ConfigSec:NewButton("Save Current Config", "Saves all current toggle/slider/color states", function()
    -- Read the name from the UI Library's internal flag table
    local name = Library.Flags["ConfigNameBox"]
    if name == "" or not name then name = "DefaultConfig" end
    Window:SaveConfig("MegaHubConfigs", name)
end)

ConfigSec:NewButton("Load Config", "Loads previously saved states", function()
    local name = Library.Flags["ConfigNameBox"]
    if name == "" or not name then name = "DefaultConfig" end
    Window:LoadConfig("MegaHubConfigs", name)
end)

-- Miscellaneous / API Example
local MiscSec = SettingsTab:NewSection("Miscellaneous")
MiscSec:NewKeybind("Panic Button", "Unloads the entire script and clears memory", Enum.KeyCode.End, function()
    Window:Unload()
end)

-- Dynamic API Example
local dynBtn
dynBtn = MiscSec:NewButton("Hide Me!", "Clicking this hides the button", function()
    dynBtn:SetVisible(false)
    Window:Notify("Element Hidden", "The button used the Dynamic API to hide itself.", 3, "Warning")
end)

-- Final Welcome Notification
Window:Notify("Welcome!", "Press RightShift to toggle the UI.", 5, "Success")
