Welcome to Kavo UI

The cleanest, most optimized UI library for Roblox exploit scripts. Built for massive hub projects requiring smooth performance and elegant design.

Getting the Loadstring

Execute this via your executor to load the module.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/sdwird/KAVO/refs/heads/main/source.lua"))()

Creating the UI

Initialize the main window. You can provide a title and a default theme name.

Lua 
local Window = Library.CreateLib("My Script Hub", "DarkTheme")

Toggling UI with Keybinds

You can easily bind a key to show/hide the entire library window.

Lua 
-- Later inside a Section
Section:NewKeybind("Toggle UI", "Hides or shows the GUI", Enum.KeyCode.RightShift, function()
    Library:ToggleUI()
end)

Applying Custom Themes

Instead of a string, pass a color table when creating your library to inject a custom theme.

Lua 
local colors = {
    SchemeColor = Color3.fromRGB(0,255,255),
    Background = Color3.fromRGB(0, 0, 0),
    Header = Color3.fromRGB(0, 0, 0),
    TextColor = Color3.fromRGB(255,255,255),
    ElementColor = Color3.fromRGB(20, 20, 20)
}

local Window = Library.CreateLib("Custom Hub", colors)

In-Game Theme Builder

Allow your users to build their own UI theme by iterating over the theme values.

Lua 
for theme, color in pairs(colors) do
    Section:NewColorPicker(theme, "Change your "..theme, color, function(color3)
        Library:ChangeColor(theme, color3)
    end)
end

Tabs

Tabs populate the left sidebar of the UI. The first tab created automatically becomes visible.

Lua 
local Tab = Window:NewTab("Main Settings")

Sections

Sections group your components together under colored headers within a Tab.

Lua 
local Section = Tab:NewSection("Player Exploits")

Updating Sections

Lua 
Section:UpdateSection("New Updated Title")

Components

The interactive elements that build out your script functionality.

Labels RichText Supported

Displays static text. Supports Roblox RichText formatting.

Lua 
local myLabel = Section:NewLabel("Status: Loading")

-- Update it later
myLabel:UpdateLabel("Status: Ready")

Buttons

A standard clickable element with an integrated tooltip icon.

Lua 
local myButton = Section:NewButton("Kill All", "Instantly kills all enemies", function()
    print("Clicked!")
end)

-- Update text
myButton:UpdateButton("Already Clicked")

Toggles

Boolean switches. You can update their text and force toggle their state from code.

Lua 
getgenv().Aimbot = false

local myToggle = Section:NewToggle("Aimbot", "Enable auto aim", function(state)
    getgenv().Aimbot = state
    print("Aimbot:", state)
end)

-- Update text only
myToggle:UpdateToggle("Silent Aim")

-- Update text AND force State to TRUE
myToggle:UpdateToggle("Silent Aim", true)

Sliders

Select numerical values. Note the parameter order: Max, then Min.

Lua 
-- (Name, Tooltip, MaxValue, MinValue, Callback)
Section:NewSlider("WalkSpeed", "Adjust your speed", 500, 16, function(s)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = s
end)

Textboxes

Input raw string text. Fires the callback when the user presses Enter or clicks away.

Lua 
Section:NewTextBox("Target Player", "Who to teleport to", function(txt)
    print("Entered:", txt)
end)

Keybinds

Assign functions to keyboard presses. Users can click to change the bind.

Lua 
Section:NewKeybind("Fly Key", "Press to fly", Enum.KeyCode.F, function()
    print("Fly executed")
end)

Dropdowns

Select an item from a list. You can dynamically refresh this list later.

Lua 
local oldList = {"Apple", "Banana"}
local newList = {"Orange", "Grape"}

local dropdown = Section:NewDropdown("Fruits", "Select a fruit", oldList, function(selected)
    print("Selected:", selected)
end)

-- Sometime later, maybe inside a button click:
dropdown:Refresh(newList)

Color Pickers

Full HSV visual color picker allowing users to customize colors.

Lua 
-- (Name, Tooltip, DefaultColor, Callback)
Section:NewColorPicker("ESP Color", "Change enemy highlight", Color3.fromRGB(255, 0, 0), function(color)
    print("New ESP Color:", color)
end)

Full Script Example

Here is a complete, ready-to-use example showcasing how to build a basic Player Hub using multiple components, tabs, and a custom toggle keybind.

Lua 
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/sdwird/KAVO/refs/heads/main/source.lua"))()
local Window = Library.CreateLib("Universal Hub", "Ocean")

-- [ 1. Main Player Tab ] --
local MainTab = Window:NewTab("Player")
local PlayerSection = MainTab:NewSection("Movement Modifications")

PlayerSection:NewSlider("WalkSpeed", "Changes how fast you walk", 250, 16, function(s)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = s
end)

PlayerSection:NewSlider("JumpPower", "Changes how high you jump", 250, 50, function(s)
    game.Players.LocalPlayer.Character.Humanoid.JumpPower = s
end)

PlayerSection:NewToggle("Infinite Jump", "Allows you to jump endlessly", function(state)
    getgenv().InfJump = state
    game:GetService("UserInputService").JumpRequest:Connect(function()
        if getgenv().InfJump then
            game:GetService("Players").LocalPlayer.Character:FindFirstChildOfClass('Humanoid'):ChangeState("Jumping")
        end
    end)
end)

-- [ 2. Visuals Tab ] --
local VisualsTab = Window:NewTab("Visuals")
local EspSection = VisualsTab:NewSection("ESP Settings")

EspSection:NewToggle("Enable ESP", "Shows players through walls", function(state)
    print("ESP State:", state)
end)

EspSection:NewColorPicker("ESP Color", "Color of the ESP highlight", Color3.fromRGB(255, 0, 0), function(color)
    print("Color changed to:", color)
end)

-- [ 3. Settings Tab ] --
local SettingsTab = Window:NewTab("Settings")
local SettingsSection = SettingsTab:NewSection("UI Configuration")

SettingsSection:NewKeybind("Toggle UI", "Hides or shows the GUI", Enum.KeyCode.RightShift, function()
    Library:ToggleUI()
end)
